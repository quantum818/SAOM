# 说唱脚本

## Getting Started
1. 第一次运行首先执行下载ffmpeg脚本。
2. 设置完成后启动。
3. 在csgo控制台输入`exec saom`。

## 命令行

在csgo聊天框中输入即可

- 点歌： saom [-s [S [S ...]]] [-m M] ，如 `saom -s 幻想万华镜op -m b` ，其中`-m b`可选，不加则为默认音源。

## Tips
- 在gui界面改动后，需要重新停止与开始。并且需要在csgo控制台中重新输入`exec saom`。
- 如果您点的音频很长（比如几十分钟），将导致音频文件很大，这种情况下您的游戏将获得高达100%的闪退几率。
